# DevOps CI/CD Node.js App

This project demonstrates a simple CI/CD pipeline using GitHub Actions.

## 🛠️ Tech Stack
- Node.js
- Express
- GitHub Actions
- Jest (for testing)

## 🚀 CI/CD Workflow
- Triggered on push and PR to `main` branch
- Installs dependencies
- Runs Jest tests

## 🔧 Run Locally

```bash
npm install
npm start
```

## 🧪 Run Tests

```bash
npm test
```

## 📂 GitHub Actions Path
`.github/workflows/main.yml`
